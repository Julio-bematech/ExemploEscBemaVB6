# ExemploEscBemaVB6
Exemplo em VB6 com comandos ESC/Bema para a MP-4200 TH, MP-5100 TH e MP-100S TH.

# Características

* Formatação básica de texto;
* Código de Barras;
* QR Code;
* Guilhotina;
* Campos editáveis e customizáveis;
Simulação de CF-e completa.
